# PURGE Engine

Procedural Universe Realtime Game Engine

## Description
PURGE is a narrative systems engine focused on canon enforcement,
rule-based validation, branching timelines, paradox handling,
and AI-assisted narrative evolution (VerseMind).

Designed for story-heavy games, simulations, and interactive fiction.

## Architecture
- core/ – Narrative logic, canon, rules, branching, integrity, AI
- ui/ – Tkinter-based control and debugging panels
- main.py – Application entry point

python main.py
